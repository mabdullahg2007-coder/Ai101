# AI Study Assistant 🤖📚

A 6-week self-taught dive into AI Engineering, built using entirely free resources — Python, applied math, core ML, LLMs, RAG, and AI agents.

## 🎯 What it does

An AI-powered study assistant that:
- Answers questions from my own course notes (digital logic, C++ OOP, applied math) with cited sources
- Generates quiz questions to test my understanding
- Tracks which topics I've covered

Built as the capstone of a 6-week self-study roadmap covering Python fundamentals, applied math, core machine learning, LLM fundamentals, prompt engineering, Retrieval-Augmented Generation (RAG), and AI agents.

## ⚙️ How it works

1. Course notes are chunked and embedded into a vector database
2. User queries are matched against relevant chunks via semantic search (RAG)
3. An LLM generates answers grounded in the retrieved content, with source citations
4. An agent layer adds quiz generation and topic-tracking on top of the core RAG pipeline
